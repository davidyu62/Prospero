# Prospero AI 투자 점수 공식 v3.0

## 개요

**Prospero AI**는 비트코인(BTC)의 투자 매력도를 **역발상(Contrarian) 전략 + 30일 추세 기반 + 신규 6개 지표**로 평가합니다.

- **점수 범위**: 0 ~ 100점
- **기본 원칙**:
  - 시장이 공포할수록 점수가 높고(매수 신호), 탐욕할수록 점수가 낮습니다(매도 신호)
  - BTC 가격이 하락해도 매크로 환경이 긍정적이면 점수가 높음 (역발상 매수 기회)
  - 최근 30일 추세와 현재값을 함께 반영
  - **v3.0**: 펀딩비, 활성주소, VIX, WTI유가, 금리차, 기대인플레이션 6개 지표 신규 추가
- **계산 주기**: 매일 1회 (UTC 05:00)
- **데이터 기간**: 최근 30일 데이터를 기반으로 추세 반영

## 1. 30일 추세 계산 방법론 (공통)

모든 "추세" 계산은 아래 방식을 적용합니다.

### 1.1 이동평균 기반 추세 판단

```
MA7  = 최근 7일 데이터의 산술평균
MA30 = 최근 30일 데이터의 산술평균

상승 추세: MA7 > MA30 × 1.005  (0.5% 이상 위)
하락 추세: MA7 < MA30 × 0.995  (0.5% 이상 아래)
중립:     그 외
```

### 1.2 변화율 계산 (30일 기준)

```
30일 변화율(%) = (가장 최근 값 - 30일 전 값) / 30일 전 값 × 100
```

### 1.3 데이터 부족 시 Fallback 처리

```
30일치 중 25일 이상 있으면      → 정상 계산
25일 미만 ~ 7일 이상           → MA7만 사용, MA30 대신 전체 평균 사용
7일 미만                       → 기본값(중립 점수) 적용
```

## 2. 크립토 지표 (55점)

### A. BTC 30일 가격 추세 역발상 점수 — 10점 (신규)

**데이터**: `btcPrice` (TB_CRYPTO_DATA)

**원칙**: BTC가 많이 하락했을수록 매수 기회. 단, 하락 속도가 너무 가파르면 위험 신호.

**계산 단계**:

**Step 1. 30일 가격 변화율 계산**
```
priceChange30d = (현재 BTC 가격 - 30일 전 BTC 가격) / 30일 전 BTC 가격 × 100
```

**Step 2. 역발상 기본 점수**

| priceChange30d | 기본 점수 | 근거 |
|---|---|---|
| < -30% | 8점 | 급락: 강한 역발상 매수 기회 |
| -30% ~ -15% | 7점 | 하락: 역발상 매수 기회 |
| -15% ~ -5% | 6점 | 약하락: 약한 역발상 기회 |
| -5% ~ +5% | 5점 | 횡보: 중립 |
| +5% ~ +20% | 4점 | 상승 중: 기회 소진 |
| +20% ~ +40% | 3점 | 강한 상승: 고점 경계 |
| > +40% | 1점 | 급등: 강한 고점 경고 |

**Step 3. 하락 속도 조정 (낙폭 과대 방지 보정)**

30일 변화율이 -30% 미만이어도, 7일 변화율이 -15% 이상이면 추가 위험으로 1점 차감.
```
7일 변화율 = (현재 가격 - 7일 전 가격) / 7일 전 가격 × 100

if priceChange30d < -30% AND priceChange7d < -15%:
    기본점수 = max(기본점수 - 1, 0)
```

**Step 4. 최종 점수**
```
A점수 = min(기본점수, 10)  ← 0~10점으로 클램핑
```

**근거**: Mean Reversion(평균 회귀) 이론. BTC는 역사적으로 30% 이상 급락 후 6~12개월 내 회복한 사례 반복. 단, 속도가 너무 빠른 경우(패닉 셀) 단기 추가 하락 위험 있어 소폭 감점.

### B. 공포탐욕지수 (Fear & Greed Index) — 20점

**데이터**: `fearGreedIndex` (TB_CRYPTO_DATA)

**원칙**: 역발상. 공포일수록 매수 기회. 30일 추세와 현재값을 결합하여 단기 과반응 완화.

**계산 단계**:

**Step 1. 30일 평균 공포탐욕지수 계산**
```
fearGreed30dAvg = 최근 30일 fearGreedIndex 산술평균
```

**Step 2. 현재값과 30일 평균 가중 결합**
```
fearGreedWeighted = 현재값 × 0.6 + fearGreed30dAvg × 0.4
```
(현재 심리에 60% 가중, 30일 추세에 40% 가중)

**Step 3. 역발상 구간 점수**

| fearGreedWeighted | 점수 | 심리 상태 |
|---|---|---|
| 0 ~ 15 | 20점 | 극도 공포: 강력 역발상 매수 |
| 16 ~ 30 | 17점 | 공포: 역발상 매수 |
| 31 ~ 45 | 13점 | 약공포: 매수 기울기 |
| 46 ~ 55 | 10점 | 중립 |
| 56 ~ 70 | 6점 | 탐욕: 경계 |
| 71 ~ 85 | 3점 | 강한 탐욕: 매도 고려 |
| 86 ~ 100 | 0점 | 극도 탐욕: 강력 매도 신호 |

**근거**: Alternative.me Fear & Greed Index는 변동성, 시장 모멘텀, 소셜 미디어, 설문, 도미넌스, 트렌드 6개 요소 가중합산. 30일 평균을 섞으면 단기 과반응을 완화하여 안정적 신호 생성.

**원칙**: 역발상. 롱이 과다하면 하락 위험, 숏이 과다하면 반등 기회.

**근거**: 파생상품 시장의 포지션 편중은 강제 청산을 유발하는 방향의 반대 신호. Binance 글로벌 롱숏비율은 가장 대표성 높은 선물 시장 지표.

### D. 오픈인터레스트 + 가격 방향 조합 — 10점

**데이터**: `openInterest`, `btcPrice` (TB_CRYPTO_DATA)

**원칙**: 가격과 OI의 조합으로 시장 구조와 건강도 판단.

**방향 판단 기준**:
```
가격 방향: (오늘 btcPrice - 7일 전 btcPrice) / 7일 전 btcPrice
OI 방향:   (오늘 openInterest - 7일 전 openInterest) / 7일 전 openInterest

상승: 변화율 > +1%
하락: 변화율 < -1%
중립: 그 외
```

**규칙** (4가지 시나리오):

| 가격 방향 | OI 방향 | 점수 | 의미 |
|---|---|---|---|
| 하락 | 하락 | 10점 | 청산 완료: 바닥 근처 신호 ✓ 매수 |
| 상승 | 하락 | 6점 | 숏커버링: 건전한 반등 신호 |
| 상승 | 상승 | 5점 | 강세 지속: 강세이나 과열 주의 |
| 하락 | 상승 | 3점 | 약세 심화: 약세 포지션 축적 |
| 중립 (데이터 불충분) | — | 5점 | 기본값 |

**근거**: Futures market microstructure theory. 가격 하락 + OI 감소 = 청산이 진행되며 시장이 정리되는 상황 → 바닥 형성 가능. 가격 하락 + OI 증가 = 약세 펀딩이 쌓이며 손실 확대.

## 3. 매크로 지표 (40점)

### E. 기준금리 (Federal Funds Rate) — 10점

**데이터**: `interestRate` (TB_MACRO_DATA)

**원칙**: 금리 수준 + 30일 추세 방향. 완화 전환 신호를 중요시.

**Step 1. 현재 수준 기본 점수**

| 기준금리 | 기본 점수 |
|---|---|
| ≤ 1.0% | 9점 |
| 1.0% ~ 2.5% | 8점 |
| 2.5% ~ 4.0% | 6점 |
| 4.0% ~ 5.0% | 3점 |
| > 5.0% | 1점 |

**Step 2. 30일 추세 방향 보정**
```
30일 전 interestRate vs 현재 interestRate 비교:

금리 하락 추세 (30일 전보다 현재가 낮음): +1점 보정 (완화 방향)
금리 상승 추세 (30일 전보다 현재가 높음): -1점 보정 (긴축 방향)
변화 없음: 0점 보정
```

**Step 3. 최종 점수**
```
E점수 = max(0, min(기본점수 + 보정, 10))  ← 0~10점 클램핑
```

**근거**: 절대 수준도 중요하지만, 방향 전환이 더 큰 시장 임팩트 생성. FED 피벗(인하 전환)은 BTC 강세의 주요 트리거.

### F. 10년물 국채금리 + 장단기 스프레드 — 8점 (신규)

**데이터**: `treasury10y`, `interestRate` (TB_MACRO_DATA)

**원칙**: 10년물 금리 수준 + 장단기 역전 여부(Yield Curve Inversion). 수익률 곡선 역전은 경기침체 강력 선행 신호.

**Step 1. 10년물 금리 수준 기본 점수**

| treasury10y | 기본 점수 |
|---|---|
| < 3.0% | 6점 |
| 3.0% ~ 4.0% | 5점 |
| 4.0% ~ 5.0% | 3점 |
| > 5.0% | 1점 |

**Step 2. 장단기 스프레드 계산 및 보정**
```
spread = treasury10y - interestRate (기준금리)

정상 수익률 곡선 (spread > +0.5%):     +1점  (경기 확장 기대)
플랫 구간 (spread 0% ~ +0.5%):       +0점  (중립)
약한 역전 (spread -1% ~ 0%):         -1점  (경기침체 경고)
강한 역전 (spread < -1%):            -2점  (경기침체 신호)
```

**Step 3. 최종 점수**
```
F점수 = max(0, min(기본점수 + 보정, 8))  ← 0~8점 클램핑
```

**근거**: 장단기 금리 역전은 1968년 이후 미국 8번의 경기침체 중 7번 선행 발생(연준 연구). 역전 폭이 클수록 위험자산 회피 심리 강화. treasury10y는 시장의 기대 인플레이션 + 장기 성장률을 반영하여 기준금리와 독립적 정보 제공.

### I. 실업률 추세 — 4점 (신규)

**데이터**: `unemployment` (TB_MACRO_DATA)

**원칙**: 절대 수준보다 30일 추세(방향)가 핵심. 실업률 상승 = 경기 악화 신호, 하락 = 완전 고용 신호.

**30일 변화량 계산**:
```
unemploymentChange = 현재 unemployment - 30일 전 unemployment (단위: %p)
```

**규칙**:

| unemploymentChange | 점수 | 의미 |
|---|---|---|
| > +0.5%p | 1점 | 급상승: 경기 악화, 단기 위험 신호 |
| 0 ~ +0.5%p | 2점 | 완만 상승: 경기 약화, FED 완화 기대 |
| 변화 없음 | 3점 | 보합: 고용 안정적 |
| -0.5%p ~ 0 | 3점 | 하락: 고용 안정, 경기 견고 |
| < -0.5%p | 4점 | 급하락: 완전 고용, 경제 강세 |

**근거**: 실업률은 전통적으로 후행 지표이나, 급격한 상승 시에는 FED 정책 전환의 선행 신호로 기능. 배점을 4점으로 제한하는 이유는 후행성 때문에 단독 신호력이 낮기 때문.

## 4. 매크로-크립토 상호작용 점수 (내장)

### 4.1 두 방향성 판단

**BTC 방향 판단 (30일 기준)**:
```
priceChange30d = (현재 BTC - 30일 전 BTC) / 30일 전 BTC × 100

BTC 상승: priceChange30d > 0%
BTC 하락: priceChange30d ≤ 0%
```

**매크로 환경 판단**:
```
macroScore = E + F + G + H + I + J  (0~40점)
macroRatio = macroScore / 40 × 100  (%)

긍정 매크로: macroRatio ≥ 55%  (40점 만점의 55% 이상 달성)
부정 매크로: macroRatio < 45%
중립 매크로: 45% ~ 55%
```

### 4.2 상호작용 점수 (Interaction Score)

| BTC 방향 | 매크로 환경 | 상호작용 점수 | 투자 시사점 |
|---|---|---|---|
| 하락 | 긍정 | **+5점** | 역발상 최고 매수 기회 (기초 체력 양호, 가격만 하락) |
| 상승 | 긍정 | +3점 | 추세 확인 매수 (모든 조건 우호적) |
| 하락 | 중립 | +2점 | 약한 역발상 기회 |
| 상승 | 중립 | +2점 | 중립적 상승 추세 |
| 상승 | 부정 | **-3점** | 경고 신호 (가격만 오르고 기초 악화) |
| 하락 | 부정 | **0점** | 중립 유지 (낙폭 과대 방지) |

**"하락 + 부정 매크로 = 0점" 설계 근거**:
음수 점수를 부여하지 않고 0점을 하한으로 설정하는 이유는, 이 구간에서도 점수가 너무 낮아 패닉 셀을 조장하지 않기 위함. 기본 공식에서 이미 하락이 반영되어 있으므로 추가 페널티 없이 중립 유지.

## 5. 최종 점수 계산

### 5.1 총합

```
totalScore = cryptoScore + macroScore + Interaction점수

cryptoScore = A + B + C + D  (0~55점)
macroScore = E + F + G + H + I + J  (0~40점)

totalScore = max(0, min(totalScore, 100))  ← 0~100 범위로 클램핑
```

### 5.2 투자 신호 판단

| 점수 범위 | 신호명 | 의미 | 색상 코드 |
|------|--------|------|---------|
| **75 ~ 100** | **Strong Buy** (강력 매수) | BTC 하락 + 긍정 매크로, 또는 전체 우호적 환경 | `#00CC33` |
| **58 ~ 74** | **Buy** (매수) | 전반적 긍정 신호, 매수 권장 | `#33DD55` |
| **38 ~ 57** | **Hold** (관망) | 혼조 신호, 현 포지션 유지 | `#FFCC00` |
| **22 ~ 37** | **Partial Sell** (부분 매도) | 부정 신호 우세, 리스크 축소 | `#FF9900` |
| **0 ~ 21** | **Strong Sell** (강력 매도) | 전체 부정 환경, 매도 강력 권장 | `#FF3333` |

## 7. 공식 근거 및 출처

### 7.1 역발상 (Contrarian) 전략 이론
- **출처**: De Bondt & Thaler (1985, Journal of Finance) - "Does the Stock Market Overreact?"
- **내용**: 투자자는 부정적 정보에 과도하게 반응하여 자산을 과매도. 이후 평균 회귀 발생.
- **적용**: 공포탐욕지수, BTC 가격 역발상, 롱숏비율

### 7.2 수익률 곡선 역전 (Yield Curve Inversion)
- **출처**: Harvey (1988), Estrella & Mishkin (1996, Journal of Political Economy)
- **내용**: 10년물 - 기준금리 역전은 12~18개월 내 경기침체의 강력한 선행지표. 연속 역전 사례 7회 중 7회 경기침체 선행.
- **적용**: 지표 F (10년물 국채금리 + 장단기 스프레드)

### 7.3 M2와 자산 가격 상관관계
- **출처**: Biais et al. (2020), Bolt et al. (2022, BIS Working Paper)
- **내용**: M2 증가 → 유동성 과잉 → 위험자산(주식, 부동산, 암호화폐) 수요 증가. BTC와 M2의 장기 상관계수 +0.7~+0.8 (2016~2022).
- **적용**: 지표 G (M2 변화율)

### 7.4 달러 인덱스와 위험자산 역상관
- **출처**: Bruno & Shin (2015, BIS Quarterly Review)
- **내용**: 강달러 환경에서 신흥국 자본 유출 및 위험자산 회피 현상. BTC-DXY 역상관 -0.6~-0.8 (2020~2023).
- **적용**: 지표 H (달러인덱스)

### 7.5 FED 반응 함수 (Taylor Rule)
- **출처**: Taylor (1993, Carnegie-Rochester Conference Series)
- **내용**: FED는 인플레이션 갭과 실업률 갭을 반응 함수로 금리 결정. CPI > 2% 지속 시 금리 인상 압박.
- **적용**: 지표 E (기준금리), J (CPI)

### 7.6 오쿤의 법칙 응용
- **출처**: Okun (1962, Brookings Papers)
- **내용**: 실업률 1%p 상승은 GDP 약 2% 하락에 대응. 실업률 급등은 경기 침체의 후행 확인이자 FED 완화 촉발 신호.
- **적용**: 지표 I (실업률 추세)

### 7.7 선물 시장의 OI-가격 다이버전스
- **출처**: Llorente et al. (2002), Mahieu & Schotman (1994)
- **내용**: 가격 하락 + OI 감소 = 청산 완료. 가격 하락 + OI 증가 = 신규 숏 진입(추가 하락 압력). 패턴은 전통 선물과 암호화폐 선물 모두에서 검증.
- **적용**: 지표 D (OI + 가격 방향)
