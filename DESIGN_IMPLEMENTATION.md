## 4. 간격 및 여백 시스템

### 정의된 간격
```
4px   - 매우 작은 간격 (아이콘 사이)
8px   - 작은 간격 (요소 사이)
12px  - 중간 간격
16px  - 기본 간격 ← standardSpacing (카드 내부)
20px  - 큰 간격
24px  - 섹션 사이
32px  - 화면 상단/하단
```

**현재 상황:**
- ✅ 기본 간격 16px (theme.standardSpacing)
- ✅ 카드 내부 padding 16-20px
- ✅ 섹션 간 spacing 20px 이상
